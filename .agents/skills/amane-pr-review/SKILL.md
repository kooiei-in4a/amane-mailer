# Amane PR Review

Read and follow:

- `AGENTS.md`
- `docs/agent-workflows/pr-review.md`

Act as Agent B.

Review only. Do not edit files, stage changes, commit, push, merge, publish artifacts, or trigger workflows.

Classify findings as:

- Blocker.
- Major.
- Minor.
- Nit.
- Question.
- Out-of-scope.

If there are no findings, explicitly say `No findings`.
